API คือ การดึงค่าข้อมูลจากเว็บไซด์อื่นๆมาเข้าในเว็บเรา อย่างเช่น เราต้องการรู้ว่าตอนนี้ที่ ม.ลาดกระบัง สภาพอากาศนั้นเป็นอย่างไร มันมีประโยชน์ตรงที่เราไม่ต้องไป copy ข้อมูลมาแก้ไขในเว็บเราบ่อยๆ มันสามารถดึงมาจากเว็บอื่นแล้วมาอัพเดทในเว็บเราทันที อย่างเว็บนี้ผมต้องการที่จะรู้ว่าสภาพอากาศ ณ ตอนนี้ที่ ม.ลาดกระบังนั้นเป็นอย่างไร ผมดึงข้อมูลมาจากเว็บ “https://developer.yahoo.com/weather/” โดยผมเอาข้อมูลสภาพอากาศบริเวณที่ Latitude ที่ "13.7540" และ Longitude ที่ "100.5014" ซึ่งเป็นพิกัดของ ม.ลาดกระบัง โดยในเว็บต้องการู้ว่า woeid ที่เราต้องการข้อมูลสภาพอากาศบริเวณนั้นอย่างเช่นที่ ม.ลาดกระบัง woeid คือ "1225448"
